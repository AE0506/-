# 微信小程序 - 校园服务平台

## 项目说明
这是一个完整的微信小程序项目，包含主页面UI和功能框架。

## 文件结构
```
├── app.json              # 小程序全局配置
├── app.wxss              # 全局样式
├── sitemap.json          # sitemap配置
├── project.config.json   # 项目配置
└── pages/
    ├── index/            # 首页
    ├── message/          # 消息页
    ├── profile/          # 我的页
    ├── search/           # 搜索页
    ├── credit-evaluation/    # 信誉评估
    ├── student-loan/     # 助学信贷
    ├── business-loan/    # 创业信贷
    ├── life-loan/        # 生活信贷
    ├── education/        # 教育公益
    ├── campus-activities/    # 校园活动
    ├── work-study/       # 勤工助学
    ├── life-payments/    # 生活缴费
    └── activity-detail/  # 活动详情
```

## 功能模块

### 主页面包含：
1. **搜索栏** - 顶部搜索功能
2. **Logo区域** - 品牌展示
3. **八大功能入口**：
   - 信誉评估
   - 助学信贷
   - 创业信贷
   - 生活信贷
   - 教育公益
   - 校园活动
   - 勤工助学
   - 生活缴费

4. **标签切换**：
   - 热门活动
   - 热门产品

5. **Banner展示**：
   - 活动宣传图
   - 参与人数统计
   - 快速入口按钮

6. **底部导航**：
   - 首页
   - 消息
   - 我的

## 使用方法

1. 在微信开发者工具中打开项目
2. 修改 `project.config.json` 中的 `appid` 为你自己的小程序ID
3. 直接编译运行，所有页面已完整实现
4. 如需自定义图片，可在对应页面替换emoji图标
5. 在 `pages/index/index.js` 中配置实际的接口地址

## 功能页面说明

### 已实现的页面：
1. **首页 (index)** - 包含所有功能入口、Banner、标签切换
2. **消息页 (message)** - 消息列表展示
3. **我的页 (profile)** - 个人信息和设置菜单
4. **搜索页 (search)** - 搜索功能和热门关键词
5. **信誉评估** - 信誉评分和详情
6. **助学信贷** - 助学贷款申请
7. **创业信贷** - 创业贷款申请
8. **生活信贷** - 生活贷款申请
9. **教育公益** - 公益活动列表
10. **校园活动** - 活动列表
11. **勤工助学** - 兼职岗位列表
12. **生活缴费** - 缴费服务入口
13. **活动详情** - 活动详细信息

## 注意事项

1. 所有页面已完整实现，可以直接运行
2. 使用emoji图标替代图片资源，无需额外准备图片
3. 如需实际功能，请在对应页面添加接口调用
4. 项目中已预留接口调用位置，可直接替换
5. 记得在 `project.config.json` 中配置正确的appid
