===============================================
    微信小程序项目导入指南
===============================================

⚠️ 重要提示：必须选择正确的文件夹！

【问题原因】
微信开发者工具提示"未找到app.json"的原因是：
你选择了错误的文件夹作为项目根目录！

【正确路径】
D:\文件汇总6\学习\自学\编程之路\Python\可运行桌面文件\程序\xxy1

【操作步骤】

方法一：直接在微信开发者工具中导入

1. 打开微信开发者工具

2. 点击"导入项目"或"+"按钮

3. 在选择目录时：
   
   【错误】❌ 不要选择：程序\
   【正确】✅ 必须选择：程序\xxy1\

   在文件选择器中：
   - 进入"程序"文件夹
   - 再进入"xxy1"文件夹
   - 选择"xxy1"文件夹
   - 此时地址栏应显示：...程序\xxy1

4. 填写项目信息：
   - AppID：选择"测试号"（如果没有正式AppID）
   - 项目名称：校园服务小程序

5. 点击"导入"

【验证】
导入成功后应该看到：
✓ 左侧文件树有 app.json
✓ 左侧文件树有 app.js
✓ 左侧文件树有 pages 文件夹
✓ 编译无错误
✓ 可以预览小程序

===============================================

方法二：最简单的办法（推荐）

如果方法一有困难，用这个超级简单的方法：

1. 找到 xxy1 文件夹

2. 复制整个 xxy1 文件夹到桌面

3. 在微信开发者工具中选择桌面的 xxy1 文件夹

4. 导入成功！

这样做的好处是路径简单，不会选错。

===============================================

【文件夹结构说明】

正确的项目结构应该是：

xxy1\                           ← 项目根目录
├── app.js                      ← 小程序入口
├── app.json                    ← 全局配置（必需！）
├── app.wxss                    ← 全局样式
├── project.config.json         ← 项目配置
├── sitemap.json                ← 站点地图
└── pages\                      ← 页面文件夹
    ├── index\                  ← 首页
    │   ├── index.js
    │   ├── index.wxml
    │   └── index.wxss
    ├── message\                ← 消息页
    ├── profile\                ← 我的页
    └── ...(其他11个页面)

【常见错误】

错误1：
选择："程序"文件夹
结果：报错"未找到app.json"
原因：app.json在xxy1文件夹里，不在程序文件夹里

错误2：
选择："xxy1云开发.py"文件
结果：无法导入
原因：这是说明文档，不是项目文件夹

正确：
选择："xxy1"文件夹
结果：成功导入！
原因：app.json就在这个文件夹里

===============================================

【快速检查】

在选择文件夹时，如果你能看到：
✓ app.json 文件
✓ app.js 文件
✓ pages 文件夹

那就选对了！点击导入即可。

===============================================

【一句话总结】

必须选择包含 app.json 的 xxy1 文件夹，
而不是它外层的"程序"文件夹！

错误：D:\...\程序\app.json         ❌ 不存在
正确：D:\...\程序\xxy1\app.json     ✅ 存在
