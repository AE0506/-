## 🚀 快速开始

### 1. 在微信开发者工具中打开
- 打开微信开发者工具
- 选择"导入项目"
- 选择 `xxy1` 文件夹
- 填写你的小程序AppID（可在project.config.json中修改）

### 2. 直接运行
项目已配置完整，可以直接编译运行，无需额外设置。

## 📝 自定义开发

### 替换emoji图标
在对应页面的`.wxml`文件中，将emoji替换为：
```xml
<image src="/images/your-icon.png" mode="aspectFit"></image>
```

### 添加实际接口
在各个页面的`.js`文件中，找到预留的接口位置，替换为实际接口：
```javascript
wx.request({
  url: 'https://your-api.com/api/endpoint',
  method: 'POST',
  data: {},
  success: (res) => {
    // 处理返回数据
  }
});
```

## ⚠️ 注意事项

1. **AppID配置**：必须在 `project.config.json` 中配置正确的小程序AppID
2. **域名白名单**：如需调用API，需要在小程序后台配置合法域名
3. **云开发**：如需使用云开发功能，请在 `app.js` 中初始化
4. **数据对接**：目前为演示版本，需要对接真实数据接口
